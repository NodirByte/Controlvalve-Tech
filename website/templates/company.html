Feihang Flow Control - ваш надежный партнер для промышленных клапанов. Мы стремимся к четкому мышлению, чтобы проектировать и производить клапаны мирового класса.

Сертифицированный по API 6D, API 600, API 602, API 607, API 6FA, ISO9001:2008, ISO 14001:2004, OHSAS 18001:2004 EAC (GOST-R), TS... Feihang разработал отличные системы контроля качества и управления, чтобы обеспечить высочайшее качество и обслуживание.

В Feihang Flow Control мы прилагаем дополнительные усилия, чтобы гарантировать качество с самого источника: сырья. Мы владеем кузнечным заводом с собственной технологией и оборудованием для производства сырья лучшего качества по конкурентной цене.

Feihang создал отличную инженерную команду. Мы сохранили и привлекли лучших специалистов в отрасли, чтобы помочь нам оставаться лидерами в области инженерии, НИОКР, поддержки проектов и послепродажного обслуживания для всех типов нужд различных клиентов на всех этапах проектов.

Feihang Flow Control создал представительства в Хьюстоне, США и Венесуэле. Их отличный послужной список дает нам уверенность в открытии новых офисов и сервисных центров в стратегических местах в ближайшем будущем, чтобы лучше обслуживать региональные потребности.

Ваши потребности - это то, ради чего мы усердно работаем. Каждый день и во всем мы в Feihang совершенствуемся, чтобы быть лучшим поставщиком и партнером в удовлетворении ваших требований и решений в отношении клапанов.
